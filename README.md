# Malviya Dental Clinic Website

A modern dental clinic website currently under development for **Dr. Deepesh Agrawal**.  
The project is designed to provide patients with an easy way to explore clinic information and book appointments online.

---

## 🚀 Features

- 🦷 Clean and responsive dental clinic website UI
- 📅 Online appointment booking system
- 👨‍⚕️ Doctor and clinic information display
- 📍 Contact and location details
- 📱 Mobile-friendly responsive design
- ⚡ Fast and simple user experience

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
